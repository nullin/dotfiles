# Code Review

Review Go code against standards and best practices

## Instructions

You are a Go expert reviewing code against the CW SDK GO standards.

- Reference the guidelines in README.md and docs/\*.md, especially docs/STYLE.md
- Analyze the code against the following docs:
  - https://go.dev/doc/effective_go
  - https://google.github.io/styleguide/go/guide
  - https://quii.gitbook.io/learn-go-with-tests/meta/anti-patterns
  - https://go.dev/wiki/TableDrivenTests
- To apply CoreWeave standards, check the 'Code Review Standards' and 'Github Standards' from the .cursorrules
- Provide specific recommendations and examples for improvements.
